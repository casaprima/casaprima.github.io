Cube pixels

¶ I started this project during Gustavo Ferreira and Joancarles Casasin’s workshop “python for (typo)graphic designers”. Drawbot provides a great environment for parametric design. I developed a script that projects parametric 3D cubes, as pixels that form bitmap font characters, in a 2D plane. Parameters are x, y, and z axis rotation, x and y perspective and spacing between cubes.
